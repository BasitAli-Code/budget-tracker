# 💰 Simple Responsive Budget Tracker

A lightweight, fully responsive client-side web application designed to help users track their income and manage daily expenses. This is one of my earliest foundational projects, built to practice responsive styling, interface layouts, and core JavaScript DOM manipulation.

👉 **[Launch Live Demo](https://basitali-code.github.io/budget-tracker)**

---

## ✨ Features & UI Highlights

* **Set Budget:** Users can input their total starting budget or income.
* **Track Expenses:** Easily add an expense with a description and amount to see the remaining balance update automatically.
* **Delete Records:** Clear individual expense items from the list to dynamically recalculate the budget.
* **Fully Responsive Design:** Engineered with fluid layouts to ensure the tracker scales cleanly and works beautifully across both mobile screens and desktop monitors.
* **Enhanced Visuals:** Integrated clean UI icons to make the action buttons and tracking lists intuitive and visually appealing.

---

## 🛠️ Built With

* **HTML5:** Structure for the budget input forms and tracking tables.
* **CSS3:** Responsive styling, media queries, and layout alignment.
* **Vanilla JavaScript:** Core logic for handling user input, mathematical calculations, and dynamic HTML updates.

---

## 💡 Key Learnings

While this project is simple, creating it helped me master the absolute basics of frontend development:
1. Capturing and validating user inputs from form fields.
2. Building fluid, responsive user interfaces that adapt to mobile and desktop screens.
3. Dynamically adding and removing HTML elements using JavaScript.
4. Performing basic math tracking to keep state balances synchronized on the screen.
